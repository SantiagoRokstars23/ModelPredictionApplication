# Objetivo

Investigar cómo medir el nivel de confianza de una predicción.

---

# Descripción

El Índice de Confianza representa qué tan sólida es una predicción según la calidad de los datos y la estabilidad del partido.

---

# Variables candidatas

- Calidad de datos
- Índice de Caos
- Diferencia de nivel
- Consistencia reciente

---

# Problema que Resuelve

Traducir múltiples indicadores en una única puntuación de confianza.
