# Modelo ELO

**Archivo:** `models/elo.md`

**Versión:** 1.0.0

**Estado:** Investigación

---

# Objetivo

Documentar el sistema de calificación ELO y evaluar su utilidad como indicador de fortaleza entre equipos.

---

# Descripción

El sistema ELO asigna una puntuación dinámica a cada equipo según sus resultados frente a rivales de distinta calidad.

---

# Problema que Resuelve

Permite estimar la fuerza relativa de dos equipos antes de un partido.

---

# Ventajas

- Fácil interpretación.
- Se actualiza continuamente.
- Considera la calidad del rival.

---

# Limitaciones

- No utiliza estadísticas avanzadas.
- No considera lesiones ni contexto del partido.
- No predice marcadores.

---

# Aplicación dentro del Modelo Santiago

Podrá utilizarse como variable complementaria para estimar la diferencia de nivel entre dos equipos.
