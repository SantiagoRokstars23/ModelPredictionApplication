# MODELO SANTIAGO
## Sistema de Predicción Probabilística para Fútbol

**Versión:** 1.0.0

**Estado:** En desarrollo

---

# 1. Objetivo

Modelo Santiago NO busca adivinar resultados.

Su objetivo es generar probabilidades estadísticamente razonables para distintos mercados de apuestas, maximizando el retorno esperado (Expected Value) mediante una correcta gestión del bankroll.

El éxito del modelo NO se mide por acertar un marcador aislado.

Se mide por:

- ROI positivo.
- Consistencia.
- Calibración de probabilidades.
- Gestión del riesgo.
- Mejora continua.

---

# 2. Filosofía

El modelo sigue cinco principios fundamentales.

## Principio 1

Los datos actuales valen más que la historia.

Un equipo será evaluado principalmente por su estado actual.

No por lo que hizo hace años.

---

## Principio 2

Nunca enamorarse del favorito.

Los favoritos suelen estar sobrevalorados por el mercado.

El modelo debe buscar probabilidades reales.

No reputaciones.

---

## Principio 3

Toda predicción debe poder explicarse.

No existen respuestas como:

"Me parece."

"Creo."

"Tengo la sensación."

Toda probabilidad deberá estar sustentada por datos.

---

## Principio 4

El modelo aprende.

Después de cada partido:

- analiza el error
- identifica la causa
- ajusta los pesos

Nunca repite exactamente el mismo análisis durante un torneo.

---

## Principio 5

La gestión del dinero es tan importante como la predicción.

Una buena predicción con mala gestión del bankroll termina perdiendo dinero.

---

# 5. Motor de Probabilidades

El modelo deberá calcular:

Probabilidad Local

Probabilidad Empate

Probabilidad Visitante

La suma SIEMPRE deberá ser:

100%

---

# 6. Motor de Marcadores

El sistema generará:

Top 4 marcadores más probables.

Cada marcador tendrá:

- porcentaje
- explicación
- nivel de confianza

Nunca se entregará un único marcador.

---

# 8. Índice de Caos

El modelo calculará qué tan impredecible es el partido.

Factores que aumentan el caos:

equipos muy parejos

muchas bajas

cambios tácticos

eliminación directa

clima extremo

rotaciones

# 9. Gestión del Bankroll

El bankroll nunca dependerá de emociones.

La inversión dependerá de:

Confianza

Índice de Caos

Valor Esperado

No de intuición.

---

# 10. Distribución de la Inversión

El modelo recomendará cómo repartir el dinero entre:

Marcador Principal + Mercado Complementario

Segundo Marcador + Mercado Complementario

Primer Marcador

Segundo Marcador

Tercer Marcador

Cuarto Marcador

La distribución podrá variar según la confianza del modelo.

# 11. Aprendizaje

Después de cada partido el sistema responderá:

¿Acertó?

¿Por qué?

¿Qué variable tuvo mayor influencia?

¿Qué variable estuvo sobrevalorada?

¿Qué peso debe ajustarse?

---

# 13. Reglas Inquebrantables

Nunca inventar información.

Nunca justificar una predicción sin datos.

Nunca cambiar un peso por intuición.

Nunca ignorar lesiones importantes.

Nunca sobrevalorar el historial.

Nunca apostar por obligación.

Si el modelo indica NO APOSTAR...

NO SE APUESTA.

---

# 14. Objetivo Final

El Modelo Santiago deberá convertirse en un sistema probabilístico capaz de adaptarse automáticamente a cualquier torneo.

No dependerá de nombres.

No dependerá de opiniones.

Dependerá únicamente de evidencia estadística.
