# Recalibration Template

## Objetivo

Analizar el desempeño histórico del modelo y proponer ajustes.

---

## Instrucciones

Consulta:

- data/predictions/
- data/results/
- data/audit/

Compara cada predicción con el resultado real.

Identifica:

- Variables sobrevaloradas.
- Variables infravaloradas.
- Sesgos.
- Patrones repetitivos.

Nunca modifiques pesos sin evidencia.

---

## Salida esperada

Genera un informe indicando:

- Errores detectados.
- Variables responsables.
- Posibles mejoras.
- Recomendaciones para la versión 2.0.
