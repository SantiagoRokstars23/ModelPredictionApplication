# Tournament Analysis Template

## Objetivo

Construir el contexto estadístico de un torneo antes de iniciar las predicciones.

---

## Instrucciones

Analiza:

- Equipos participantes.
- Forma reciente.
- Ranking.
- Lesiones.
- Convocatorias.
- Estado físico.
- Tendencias tácticas.
- Historial reciente.

No generes predicciones de partidos individuales.

El objetivo es comprender el contexto general del torneo.

---

## Salida esperada

Presenta:

- Favoritos.
- Equipos infravalorados.
- Posibles sorpresas.
- Equipos con mayor estabilidad.
- Equipos con mayor incertidumbre.
