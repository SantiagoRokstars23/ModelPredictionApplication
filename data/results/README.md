# Results

**Directorio:** `data/results/`

**Versión:** 1.0.0

**Estado:** Activo

---

# Objetivo

Registrar los resultados oficiales de cada partido.

Estos datos constituyen la verdad histórica utilizada para evaluar el modelo.

---

# Responsabilidad

Almacenar resultados verificables provenientes de fuentes oficiales.

---

# Contenido

Ejemplos:

- resultados.csv
- estadisticas-finales.csv

---

# Reglas

Nunca modificar resultados oficiales.

Toda información deberá indicar su fuente.

Los resultados deberán conservarse permanentemente.

---

# Consumidores

- Auditor
- Statistician
