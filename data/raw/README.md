# Raw Data

**Directorio:** `data/raw/`

**Versión:** 1.0.0

**Estado:** Activo

---

# Objetivo

Almacenar la información obtenida directamente desde las fuentes externas.

Representa la copia original de los datos.

Ningún archivo contenido en este directorio deberá modificarse manualmente.

---

# Responsabilidad

Este directorio actúa como la fuente primaria de información del Modelo Santiago.

Su propósito es conservar evidencia del origen de todos los datos utilizados.

---

# Contenido

Ejemplos:

- partidos.csv
- jugadores.csv
- lesiones.csv
- cuotas.csv
- rankings.csv

---

# Reglas

Nunca editar archivos manualmente.

Nunca eliminar archivos históricos.

Nunca utilizar directamente esta información para generar predicciones.

Toda información deberá pasar por el proceso de validación.
