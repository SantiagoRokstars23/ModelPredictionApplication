# Predictions

**Directorio:** `data/predictions/`

**Versión:** 1.0.0

**Estado:** Activo

---

# Objetivo

Almacenar todas las predicciones generadas por el Modelo Santiago.

Cada predicción constituye evidencia histórica del comportamiento del modelo.

---

# Responsabilidad

Registrar las probabilidades calculadas antes del inicio de cada partido.

---

# Contenido

Ejemplos:

- predictions.csv
- world-cup-2026.csv
- copa-america-2028.csv

---

# Reglas

Nunca sobrescribir predicciones.

Nunca modificar una predicción después del inicio del partido.

Toda predicción deberá incluir la versión del modelo utilizada.
